API 仕様書

ベース URL: /api

認証

Google OAuth リダイレクト

GET /api/auth/google

レスポンス (200):

{
  "url": "https://accounts.google.com/o/oauth2/auth?..."
}

Google OAuth コールバック

GET /api/auth/google/callback?code=...&state=...

成功時: FRONTEND_URL/auth/callback へリダイレクト
失敗時: FRONTEND_URL/auth/error へリダイレクト

認証済みユーザー取得

GET /api/auth/user

認証: 必要

レスポンス (200):

{
  "user": {
    "id": 1,
    "name": "John Doe",
    "email": "john@example.com",
    "avatar_url": "https://lh3.googleusercontent.com/..."
  }
}

レスポンス (401):

{
  "error": "認証が必要です"
}

ログアウト

認証: 必要

レスポンス (200):

{
  "message": "ログアウトしました"

QR コード生成 (同期)

POST /api/qrcodes
Content-Type: application/json

リクエスト:

{
  "data": "https://example.com"
}

バリデーション: data - 必須, 文字列, 最大 1000 文字

レスポンス (201):

{
  "message": "QRコードを生成しました",
  "qrcode": {
    "id": 1,
    "user_id": 1,
    "file_name": "1_1705318200_507f1f77.png",
    "data": "https://example.com",
    "status": "completed",
    "created_at": "2024-01-15T10:30:00.000Z",
    "updated_at": "2024-01-15T10:30:00.000Z",
    "url": "http://localhost:9000/qrcodes/1_1705318200_507f1f77.png"
  }
}

レスポンス (422):

{
  "error": "バリデーションエラー",
  "messages": {
    "data": ["data は必須です"]
  }
}

    "status": "pending",
    "data": "https://example.com",

メール

メール送信

POST /api/mail/send
Content-Type: application/json

認証: 必要

リクエスト:

{
  "to": "user@example.com",
  "subject": "テスト件名",
  "message": "テスト本文"
}

バリデーション:

  • to - 必須, 有効なメールアドレス
  • subject - 必須, 最大 255 文字
  • message - 必須, 最大 5000 文字

レスポンス (200):

{
  "message": "メールを送信しました"
}
